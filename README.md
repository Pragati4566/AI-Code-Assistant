This is a **full-stack AI Code Review application** built using  
**React (Frontend)** and **Node.js + Express (Backend)**.

---

##  Features

- Full-stack architecture (Frontend + Backend)
- React-based user interface
- Express.js REST API
- AI-powered code review logic
- Clean and scalable project structure
- Easy to extend with real AI models (Gemini / OpenAI)

---

##  How the Application Works (Simple Explanation)

1. User enters code in the React frontend
2. Frontend sends the code to backend using an API request
3. Backend processes the input using AI logic
4. AI-generated review is sent back as a response
5. Frontend displays the review to the user

 Communication between frontend and backend happens via **REST APIs**
